Cloud Sync Setup

ℹ️ Cloud Synchronization
This app can sync your data across all devices in real-time using Firebase (Google's free cloud database). Set it up once, and access your data from anywhere!

Step 1: Create a Firebase Project

  1. Go to Firebase Console
  2. Click "Add Project" and give it a name (e.g., "Equipment-Tracker")
  3. Disable Google Analytics (not needed) and click "Create Project"
  4. Once created, click the Web icon (</>) to add a web app
  5. Give your app a name and click "Register app"
  6. Copy the Firebase configuration object

Step 2: Enable Firestore Database

  1. In Firebase Console, go to "Firestore Database" in the left menu
  2. Click "Create Database"
  3. Choose "Start in test mode" (for development) and click "Next"
  4. Select your preferred location and click "Enable"
⚠️ Security Note: Test mode allows unrestricted access for 30 days. For production, you'll want to set up proper security rules.

🛠️ Troubleshooting

If you're experiencing issues with equipment status not updating (like items stuck as "Out of Service"), try clearing the cache:

⚠️ If using local storage mode, this will delete ALL data. If using Firebase, data will reload from the cloud.

⚠️ Data not showing after login? Your Firestore Security Rules may be blocking reads. Go to Firebase Console → Firestore → Rules and make sure your rules allow authenticated users to read. The simplest working rule while you set up roles: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
Paste this in Firestore Rules and click Publish, then refresh this page and sign in again.
